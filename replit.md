# Python Project

## Overview
A simple interactive Python application that demonstrates basic Python functionality.

## Features
- Interactive greeting system
- Simple calculator that performs basic arithmetic operations (addition, subtraction, multiplication, division)
- Input validation and error handling

## Recent Changes
- November 26, 2025: Initial project setup with Python 3.11
- Created interactive Python application with greeting and calculator features

## Project Architecture
- `main.py`: Main application file with interactive console interface
- Uses Python 3.11
- Console-based application with user input

## How to Run
The application runs automatically via the configured workflow. It will:
1. Ask for your name
2. Greet you
3. Run a simple calculator that accepts two numbers and shows all basic operations
